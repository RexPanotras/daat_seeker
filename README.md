DA'AT Seeker（知识追寻者）是一个基于Flask的Web应用，用于计算和查找数学和科学知识。

## 功能
- 计算调和级数
- 寻找调和级数和超过目标值的最小n
- 计算洛伦兹变换
- 计算1~n的和

## 如何使用

### 环境搭建
1. 确保安装了Python 3.7+（推荐使用3.10-3.12版本）和Node.js 14+
2. 克隆仓库到本地

### 后端服务
1. 进入backend目录：`cd backend`
2. 安装依赖：`pip install -r requirements.txt`
3. 启动服务：`python run.py`
4. 后端服务将在 http://localhost:8086 运行

### 前端服务
1. 进入frontend目录：`cd frontend`
2. 安装依赖：`npm install`
3. 启动服务：`npm run dev`
4. 前端服务将在 http://localhost:5173 运行（如果端口被占用，会自动使用其他端口）

### 访问应用
在浏览器中打开前端服务的地址（默认为 http://localhost:5173），即可访问DA'AT Seeker应用。

### 使用功能
1. **计算调和级数**：输入一个非负整数n，点击"计算"按钮，即可得到调和级数的和。
2. **寻找调和级数和超过目标值的最小n**：输入一个目标值，点击"计算"按钮，即可得到调和级数和超过该目标值的最小n。
3. **计算洛伦兹变换**：输入x、t和v值，点击"计算"按钮，即可得到洛伦兹变换的结果。
4. **计算1~n的和**：输入一个非负整数n，点击"计算"按钮，即可得到1到n的和。如果输入值过大导致溢出，会显示"溢出"提示。
